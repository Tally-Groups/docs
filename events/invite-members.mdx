After you [create an event](/events/create-event), invite members so they receive an email and can RSVP (and pay if the event has a fee).

## Add invitees

1. Open the event and go to the **Invite** or **Invitees** section.
2. Add members:
   - **By group**: Select one or more [groups](/groups/using-groups-for-events). Everyone in the group gets an invite.
   - **Individually**: Search or select members and add them.
3. For **paid events**, you can set or waive the **amount** per invitee. See [Event payments](/events/event-payments).

## Send invites

1. After adding invitees, click **Send invites** (or **Send**).
2. Rowhouse sends an email to each invitee with:
   - Event details (title, date, location)
   - A link to **RSVP** (and pay if applicable)
3. Invitees appear in the event with status **pending** until they RSVP. You can track [RSVP](/events/rsvp) and [attendance](/events/attendance) on the event page.

<Tip>
Send invites soon after creating the event so members have time to respond. You can send reminders later. See [Reminders](/events/reminders).
</Tip>

## Add or remove invitees later

- **Add**: Add more members or groups and click **Send invites** again. New invitees receive the email; existing ones are not re-sent unless you use "Resend" (if available).
- **Remove**: Remove a member from the invitee list. They will no longer see the event in their invite flow; any existing RSVP or payment may remain in history for your records.

<Note>
If you use [calendars](/events/calendars) with calendar members, assigning the event to a calendar may auto-add those members as invitees. Check the invitee list and send invites when ready.
</Note>
