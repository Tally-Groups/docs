Create an event so you can invite members, track RSVPs, and optionally collect payment.

## Create a new event

1. Go to **Events** in the sidebar and click **Create event** (or **New event**).
2. Fill in:
   - **Title**: e.g. "Fall Kickoff Meeting"
   - **Description**: optional details (agenda, what to bring)
   - **Start date and time** (and optionally **End time**)
   - **Location**: address or place name
   - **Amount** (optional): leave blank for a free event, or enter a number (e.g. 10 for $10) for a paid event. Currency is usually USD unless set at org level.
   - **Calendar** (optional): assign to a [calendar](/events/calendars) if you use them
3. Save or **Create**. The event appears in the Events list and you can [invite members](/events/invite-members) next.

<Tip>
For paid events, set the amount before sending invites so members see the correct price when they open the pay link.
</Tip>

## Free vs paid

- **Free**: Leave **Amount** blank or zero. Invitees only RSVP; no payment step.
- **Paid**: Set **Amount** (in dollars or cents depending on the UI). When you assign invitees, they can pay via the event pay link. You can [waive](/events/event-payments) for specific people.

## After creating

- [Invite members](/events/invite-members) (individually or by [group](/groups/using-groups-for-events)).
- Share the [public event page](/events/public-event-page) or add to an [iCal feed](/events/ical).
- Set up [reminders](/events/reminders) if your plan supports them.
