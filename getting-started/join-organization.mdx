You can join an existing Rowhouse organization in two ways: **invite link** or **join code**.

## Join via invite link

1. An admin sends you an **invite link** (e.g. `rowhouse.co/clubs/invite?token=...` or `rowhouse.co/clubs/join` with context).
2. Open the link in your browser.
3. If you're not signed in, sign in (or sign up) with the email the admin used for the invite.
4. Accept the invite. You'll be added as a member and can switch to that organization in the sidebar.

<Tip>
Use the same email address the admin used when inviting you, or the invite may not match your account.
</Tip>

## Join via join code

1. Sign in to Rowhouse.
2. Open the **club switcher** in the sidebar (top of the sidebar) and click **Join Club**.
3. Enter the **join code** your admin shared (e.g. a short alphanumeric code).
4. Submit. You'll be added to that organization.

<Note>
Join codes are set by admins in [Club settings](/settings/club-settings). If the code doesn't work, ask an admin to confirm it and your email.
</Note>

## After joining

- Your **role** is usually **member** by default. Admins can change it to **admin** if you need to manage the org. See [Roles and permissions](/members/roles-and-permissions).
- Switch between organizations using the **club switcher** in the sidebar. See [Switching clubs](/getting-started/switching-clubs).

If you don't see the organization or the invite fails, see [Troubleshooting](/getting-started/troubleshooting).
