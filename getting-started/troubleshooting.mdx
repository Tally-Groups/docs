Common issues when signing up, logging in, or onboarding your organization.

## Sign-up and login

### I didn't receive the verification email

- Check **spam** and **promotions** (or other tabs).
- Add the sender to your contacts and request the verification email again from the sign-up flow.
- Ensure you entered the correct email address.

### I can't log in with Google

- Use the **same Google account** you used when you first signed up for Rowhouse.
- If you signed up with email and later try Google, Rowhouse may treat that as a different account. Use **Log in with email** and your password, or use the same Google account everywhere.

### I forgot my password

- On the login page, use **Forgot password?** and enter your email. Use the reset link from the email to set a new password. Links expire; request a new one if needed.

### I don't see my organization after logging in

- Use the **club switcher** at the top of the sidebar. Your org might be in the list; select it.
- If you were invited, ensure you accepted the invite and used the same email the admin invited.
- If you just created the org, refresh the page or log out and back in. If it still doesn't appear, try creating again or contact support.

## Onboarding and Stripe

### Onboarding is stuck or Stripe won't complete

- Finish **all required fields** in Stripe's screens (business type, representative, EIN if required, bank account).
- Use a supported browser and disable ad blockers or extensions that might block Stripe.
- If Stripe shows an error, follow their message (e.g. fix invalid routing number). Then return to Rowhouse and try connecting again.

### I need to add or change my bank account

- Go to **Settings** (or **Subscription** / **Stripe**) and use **Add bank account** or the link to Stripe. You can also add or update bank details in your [Stripe Dashboard](https://dashboard.stripe.com).

### Payments aren't working / "Connect Stripe" still showing

- Complete [Stripe Connect](/getting-started/stripe-connect) and any steps Stripe asks for (identity, bank). Until Connect is complete, members cannot pay.
- If you've already completed Stripe, refresh the page or log out and back in. If the dashboard still shows incomplete, try re-opening the Stripe connection from Settings.

## Join code and invites

### Join code doesn't work

- Confirm the code with an **admin** (no extra spaces, correct characters). Admins can see or reset the join code in [Club settings](/settings/join-code-and-invite).
- Ensure you're signed in with the email the admin expects, if applicable.

### I was invited but don't see the organization

- Check that you **accepted the invite** (e.g. clicked the button in the invite email or page).
- Open the **club switcher** and look for the org in the list.
- Ensure you're logged in with the account that received the invite.
