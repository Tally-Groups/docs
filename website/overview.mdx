The **Website** section lets you create and edit **pages** for your organization's public site, view **analytics**, and (if supported) set a **custom domain**.

## What you can do

- **Pages**: Create and edit pages with a drag-and-drop [page editor](/website/page-editor) (Puck). Each page has a **slug** (URL path), **title**, and optional **layout** (wrapper for child pages).
- **Publishing**: Pages can be **draft** or **published**. Only published pages are visible on the public site. See [Publishing](/website/publishing).
- **Analytics**: View [website analytics](/website/analytics) (e.g. page views, traffic) if your plan includes it.
- **Custom domain**: Point your own domain (e.g. www.myclub.com) to your Rowhouse site if the feature is available. See [Custom domain](/website/custom-domain).

## Where to find it

- Open **Website** in the sidebar (under the main nav or its own section). You will see **Pages** and **Analytics**. Website features may require a **paid plan**. See [Plan limits](/guides/plan-limits).

## Typical flow

1. Go to **Pages** and [create a page](/website/pages) (title, slug).
2. Open the [page editor](/website/page-editor) and add content (sections, text, images, buttons).
3. [Publish](/website/publishing) the page so it goes live.
4. Share the URL (e.g. `rowhouse.co/your-org-slug/about`) or set a [custom domain](/website/custom-domain).

See [Troubleshooting](/website/troubleshooting) for page not showing, slug conflicts, or editor issues.
