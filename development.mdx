This documentation site is built with [Mintlify](https://mintlify.com). Use the following to edit and preview docs locally.

## Run the docs locally

1. Install the Mintlify CLI:
   ```bash
   npm i -g mint
   ```
2. From the `docs` directory, run:
   ```bash
   mint dev
   ```
3. Open **http://localhost:3000** in your browser. The preview updates as you edit MDX files.

<Tip>
Keep the terminal open while editing; changes are reflected automatically.
</Tip>

## Edit content

- **Pages** are MDX files (e.g. `members/overview.mdx`). Use standard Markdown and [Mintlify components](https://mintlify.com/docs/components) such as `<Card>`, `<Tip>`, `<Accordion>`, and `<CodeGroup>`.
- **Navigation** is defined in `docs.json` under `navigation.tabs[].groups`. Add or reorder `pages` entries to change the sidebar.

## Check links

From the `docs` directory, run:

```bash
mint broken-links
```

This reports broken internal links so you can fix them before deploying.

## Deploying

Deployment depends on your setup (e.g. Mintlify GitHub app, or your own CI). Commit and push changes to the branch that triggers your docs deployment.
