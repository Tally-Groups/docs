Control how new members **join** your organization: **join code** (for existing Rowhouse users) and **invite link** (for sharing by email or social).

## Join code

- The **join code** is a short code (e.g. alphanumeric) that someone enters in Rowhouse when they click **Join Club** (from the club switcher). Only people who know the code can join this way.
- **Set or change**: In **Settings** (or **Club settings**), find **Join code** (or **Invite code**). Enter or generate a code and save. Share it only with people you want to join (e.g. members, recruits).
- **Reset**: Change the code if it was shared too widely or compromised. The old code stops working; share the new code with the right people.

<Note>
Join code is optional. If left blank, joining may be by invite link only (if supported) or by admin-adding members.
</Note>

## Invite link

- The **invite link** is a URL that you share (e.g. in email, social). When someone opens it, they can sign up or sign in and **accept the invite** to join your organization. The link may be generic (e.g. `rowhouse.co/clubs/join`) or include a token; behavior depends on the app.
- **Get the link**: In **Settings** or **Members**, look for **Invite link**, **Share invite**, or **Invite members**. Copy the URL and share it. You may be able to customize the message or see how many have joined via the link.

<Tip>
Use the invite link for recruitment (e.g. "Join us: [link]") and the join code for existing Rowhouse users who you have already told the code.
</Tip>

## Who can join

- Anyone with the **join code** (and a Rowhouse account) can join by entering it.
- Anyone with the **invite link** can open it and join (they may get an account created if they sign up with email). Restrict sharing so only intended people join.
